My practice

Back-end part:

  • At first we plugged in ssd, connected power and made physical server work. I found out about raid controller and how server works from inside.
  • After getting server to work, I installed Proxmox VE(Virtual environmet). Then, it came to IP-configuration, so I created a local network and putted my virtual environment into it, so my future server could be available in that loacl network.
  • Next step was installing operating system. We used remote controll to install and set up 'Ubuntu server' operating system. Then, it came to IP-configuration again, but for server that time.
  • Last crutial part was setting up LAMP stack, which stays for Linux Apache MySql PHP configuration for web server. Then we run a website and it end up in front-end practice

Front-end part:

  • I started my front-end with making a scetch in figma online editor.
  • After, I started to copy my figma design into Wordpress editor, using plugins such as Calibri and Elementor
  • After I got my website ready and functioning using Wordpress, I stated to copy figma design using html/CSS combination
